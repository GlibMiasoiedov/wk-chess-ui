# WK Chess UI

**WK Chess UI** is a specialized WordPress plugin designed to power the interactive chess learning experience at [White Knight Academy](https://whiteknight.academy/).

## Project Overview

This repository contains the source code for the interactive chess interface used by White Knight Academy to deliver high-quality **online chess lessons**. Built with React and Vite, it ensures a smooth and responsive experience for students who want to **learn chess online**.

The plugin seamlessly integrates into WordPress, providing a dedicated "Arena" and learning modules where students can practice, solve puzzles, and participate in **chess classes online**.

## Key Features

*   **Interactive Chess Board:** Powered by `cm-chessboard` and React for a responsive user experience.
*   **Integrated Learning:** Custom modules designed for effective **online chess lessons**.
*   **Performance:** Optimized build using Vite for fast loading times.
*   **WordPress Integration:** Easy embedding via shortcodes and page templates.

## Development

### Prerequisites
*   Node.js & npm
*   PHP & WordPress environment

### Setup
1.  Navigate to the `ui` directory: `cd ui`
2.  Install dependencies: `npm install`
3.  Run development server: `npm run dev`

### Build
To build the project for production:
`npm run build`

The output will be generated in the `dist` folder, ready to be enqueued by the WordPress plugin.
